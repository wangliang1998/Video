# Video
基于前后端分离技术的视频点播平台

# 摘要
随着互联网技术和多媒体技术的高速发展，以视频点播为主的网络视频成为人们日常娱乐的重要方式。然而，当前的视频点播平台中的大多视频都不能免费观看，而且还有广告，用户体验不好。

本系统旨在使用前后端分离技术设计并实现一个视频点播平台，通过爬虫技术从互联网上采集视频信息并保存到数据库。在此平台上所有视频都可免费观看，而且没有广告，并且支持多端自适应、跨平台点播。

在技术方面，前端使用Vue框架开发的Web平台作为用户体验平台，后端使用Spring Boot框架进行开发。在信息安全方面，采用JWT技术用于用户身份认证以及权限控制。在数据交互方面，后端提供RESTFUL接口，前端使用axios向服务器接口发生请求来获取json数据。

关键词：前后端分离；视频点播平台；Spring Boot；爬虫；浏览器/服务器模式

## 各模块功能介绍如下：

- 数据采集与存储：通过爬虫技术在互联网上采集视频基本信息，然后将数据保存到数据库。
- 用户身份认证：在前端，只有用户成功登录后才能访问除登录页面外的其他页面。在后端，服务端程序只响应前端发送的携带正确token信息的请求。
- 权限控制：在前端，只有管理员才能访问后台管理页面。在后端，服务器程序只会响应给普通用户获取视频信息的请求，但会响应管理员所有的请求。
- 视频播放：对视频进行在线播放。
- 登录：用户登录成功后，后端返回给前端身份标识信息token，之后前端的任何请求都必须在请求头上加上token。
- 视频信息提供：提供视频信息相关数据接口，返回JSON格式数据。包括提供电影信息、电视剧信息、综艺信息三大类接口。
- 视频搜索：根据搜索关键字返回相关的视频信息。
- APP下载：用户访问该接口地址时，服务器会发送APP文件数据，前端会提示进行下载。
- 用户管理：主要对用户信息进行查看以及编辑。
- 基本信息维护：主要对电影类别信息和解析接口信息进行维护。
- 视频信息管理：主要用于管理员添加视频信息、对视频信息进行维护以及查看视频信息。
