## 基于前后端分离技术的电影点播平台
- 用户模块：用户注册、登录、退出、信息修改
- 电影模块：电影信息分页查看、分类查看、搜索查看、信息修改
- 播放模块：电影播放
- 爬虫模块：电影信息抓取和保存
- 解析接口模块：解析接口增删改查
- 后台管理模块：进行信息的维护

### 前端技术：Vue+Element+Axios
### 后端技术：SpringBoot+MySQL+Redis+Mybatis+Jsoup
